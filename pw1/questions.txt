PW1-A
Did you observe any differences when changing the order of the lines manipulating the transformations?

No, we did not observe any differences when changing the order of the lines manipulating the transformations. 
This is because, in Three.js, each object's local transformation matrix is automatically updated from its position, rotation and scale properties in a fixed order: M = T * R * S (translation * rotation * scale).
Differences would only appear if we disabled this behavior by setting the matrixAutoUpdate property to false and manually updated the transformation matrix ourselves, where the multiplication order would then matter.

PW1-B
What were the differences between setting the rotation property and calling the rotateX() function? Did it behave as it would in WebGL/WebCGF?

When setting the rotation property twice, the object behaved the same as if we had only had it set once, the second assignment simply overwrote the first.
However, when using the rotateX() function twice, the rotations accumulated, so the object actually rotated twice.
This behavior matches what happens in WebGL.

PW1-C

How does the shininess affect the material components (ambient, diffuse, specular)?

- The shininess affects the specular component of light reflection, controlling the size and sharpness of highlights. The ambient and diffuse components, are not directly affected by shininess.

What were the visual differences between changing the point or the ambient light to red?

- Changing the point light to red added a specific directed red light with shadows and highlights, but when we changed the ambient light to red it adds a general uniform red tint to the whole scene, with no direction or shadows.

- When the light source was moved from (0, 20, 0) to (0, â€“20, 0), the entire scene appeared in a red shadow, since the location of the point light was far below the scene.
This is related to the local illumination model, which combines three components: ambient, which is independent of light or viewer position; diffuse, which depends on the angle between the light direction and the surface normal; and specular, which depends on the viewer position and the reflection direction.
When the light was overhead, the diffuse and specular terms were significant, but when it was moved below the scene, only the ambient term remained, since the others were zero or negative.
